# BCC266---TP2

Trabalho Prático 2 de Organização de Computadores - UFOP

## Compilar e rodar

### Manualmente

```bash
  make
  ./gerador_instrucoes # para gerar o arquivo instructions2.txt
  make run
```

### Script de testes

```bash
  ./rodar_testes.sh
```

## Instruções Makefile

para compilar:

```bash
  make
```

para rodar e compilar se necessário:

```bash
  make run
```

para apagar arquivos .o:

```bash
  make clean
```
